# 🛒 ecommerce-academic-millanel

Simulación de tienda online de perfumes que implementa fielmente el **Registro PERFUME** con clave primaria, rangos de fecha y validaciones. Proyecto académico desarrollado en HTML, CSS y JavaScript puro.

## 📌 Descripción

Aplicación web funcional que permite:
- Visualizar un catálogo de perfumes (4 ejemplos iniciales).
- Agregar productos al carrito, modificar cantidades, eliminar ítems y vaciar el carrito.
- Simular una compra con modal de confirmación y descuento de stock.
- Añadir nuevos perfumes mediante un formulario que respeta:
  - **Clave primaria única** (`Código` entero).
  - Rango de `Día` (1..31), `Mes` (1..12) y `Año` de 4 dígitos según el registro.

## 🚀 Tecnologías

- HTML5 semántico y responsivo.
- CSS3 (Flexbox, Grid, diseño tipo tarjetas).
- JavaScript ES6 (manejo de estado, manipulación del DOM, validaciones).
